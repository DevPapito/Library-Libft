## Philosophy

The philosophy of the project follows the same as School 42, understanding the foundation of things, where we see what is under the hood and learn, even in the difficulty to understand the main software concepts.

This compilation of functions in a library allows at first sight a beginner in software development to carry out a medium sized project with great technological appeal.

The technologies employed being C, low level libs, Makefile and shell allow the developer to carry out development with necessary limitations to maintain a cohesive programming rich in SOLID concepts

May 26 - 18:50
Maked 42 functions using low-level library Unistd.h

The "_fd" is meaning File Descriptor, the function using file descriptors to write in personalized file descriptor

* putchar_fd
Put a char in respective file descriptor

* putstr_fd
Put a pointer of chars in respective file descriptor

* putendl_fd
Make equal putstr_fd but add new line with char "'\n'" in end file

* putnbr_fd
Transform with recursive method numbers integer to equals but in ascii and put in respective file descriptor
