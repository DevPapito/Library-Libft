## Filosofia
A filosofia do projeto segue a mesma da Escola 42, entender o fundamento das coisas, aonde nós vemos o que está de baixo do capó e aprendemos, mesmo na dificuldade a entender os principais conceitos de software.

Esse compilado de funções em uma biblioteca permite de primeira vista um iniciante no desenvolvimento de software realizar um projeto de médio porte e de grande apelo tecnológico.

As tecnologia empregas sendo elas C, libs de baixo nivel, Makefile e shell permitem que o desenvolvedor realize o desenvolvimento com limitações necessáriar para manter uma programação coesa e rica nos conceitos de SOLID
